## 🎯 优化建议（按页面）

### 📱 **页面 1: 首页 (思格-首页)**

#### 当前问题：
- ❌ 卡片缺乏层次感（仅有边框，无阴影）
- ❌ AI 洞察区域虽然边框为橙色，但不够突出
- ❌ 卡片之间的垂直节奏感不强
- ❌ 底部导航栏的图标可以更精致

#### 优化方案：

**① 卡片添加微阴影效果**
```
目标节点: 2:11, 2:15, 2:175
效果类型: DROP_SHADOW
参数设置:
  - 颜色: rgba(0, 0, 0, 0.5)
  - 偏移: X=0, Y=4
  - 模糊半径: 12px
  - 可见性: true
目的: 增加卡片与背景的分离度，提升层次感
```

**② AI 洞察区域增强**
```
目标节点: 2:175
修改内容:
  A. 背景: 从纯色改为渐变
     - 起点: rgba(26, 26, 26, 0.9) (90%透明度的卡片色)
     - 终点: rgba(91, 52, 37, 0.15) (15%透明度的深橙色)
     - 方向: 从上到下线性渐变
     目的: 让 AI 区域在视觉上更突出，暗示"智能/温暖"

  B. 阴影增强:
     - 颜色: rgba(232, 132, 95, 0.2) (20%透明度的珊瑚色)
     - 偏移: X=0, Y=6
     - 模糊半径: 16px
     目的: 橙色光晕让 AI 区域更吸引注意力
```

**③ 卡片间距调整**
```
当前状态: 卡片之间间距约 12-16px
建议调整: 增至 20px
原因: 更大的间距让内容更透气，减少拥挤感
具体操作:
  - 主内容区域 (2:5): 子元素之间 padding/margin 增加 4-8px
  - 或者给每个卡片添加 margin-bottom: 16px
```

**④ 底部导航图标优化**
```
当前: 文字+简单图标
建议:
  - 选中状态的图标添加微光效果 (inner glow)
  - 未选中状态使用 #9CA3AF (次要文字色)
  - 选中状态使用白色 + 橙色底部指示条 (3px 高度)
```

### ✏️ **页面 2: 新建笔记 (思格-新建笔记)**

#### 当前问题：
- ⚠️ 输入框缺乏焦点反馈（用户不知道哪个输入框处于激活状态）
- ⚠️ 工具栏图标较小，点击区域可能不够大（移动端推荐最小 44x44px）
- ⚠️ "保存"按钮虽然使用了橙色渐变，但缺少立体感

#### 优化方案：

**① 输入框焦点状态**
```
目标节点: 标题输入框 (推测为 2:48 附近), 内容输入框
建议添加效果:
  - 默认状态: 边框颜色 #2A2A2A (暗灰)
  - 焦点状态: 边框颜色变为 #D4724A (橙色) + 微弱外发光
    发光参数: rgba(212, 114, 74, 0.3), blur: 8px
实现方式: 在 Ardot 中创建 Component Variants (Default / Focused)
```

**② 工具栏增强**
```
目标节点: 底部工具栏区域 (推测为 2:60 或其子节点)

尺寸优化:
  - 图标容器大小: 从当前的 ~40px 增至 48px (符合 iOS HIG 标准)
  - 图标本身: 保持 24px，周围增加 padding

视觉优化:
  - 为工具栏添加半透明背景:
    背景: rgba(26, 26, 26, 0.9)
    模糊效果: backdrop-filter: blur(10px) (毛玻璃效果)
  - 图标颜色: 未激活 #666666 → 激活 #FFFFFF
```

**③ 保存按钮升级**
```
目标节点: 2:49 (保存按钮)

当前: 线性渐变填充 (#E8845F → #D4724A)
建议增加:
  A. 阴影效果:
     - 类型: DROP_SHADOW
     - 颜色: rgba(212, 114, 74, 0.4)
     - 偏移: X=0, Y=4
     - 半径: 12px
     效果: 按钮悬浮感，鼓励点击

  B. 按压状态 (Pressed State):
     - 缩放: scale(0.98)
     - 阴影减弱: opacity 降低至 0.2
     目的: 提供触觉反馈的视觉替代
```

- ⚠️ 页面下方空白较多，未充分利用空间

**① 用户信息卡片升级**
```
目标节点: 2:106

当前结构: 头像(圆形渐变) + 名字 + 简介
建议增加:
  A. 头像光环效果:
     - 在头像外侧添加一个淡淡的圆形光晕
     - 颜色: rgba(232, 132, 95, 0.2)
     - 大小: 比头像大 16px
     目的: 让头像更突出，增加品牌识别度

  B. 背景纹理:
     - 在卡片右上角添加一个抽象的网格图案或斜线图案
     - 不透明度: 5%
     - 目的: 打破纯色的单调感
```

**② 统计数据可视化**
```

  - 每个统计项上方添加对应的小图标 (SVG, 24px, 橙色)
  - 数字字体: 使用 Semibold/Bold 字重，增大 2-4px

图标建议:
  - 笔记: document/text icon
  - 标签: tag/hash icon
  - 天: calendar icon
```

**③ 功能菜单区域**

```
新增内容: 在统计数据下方添加功能入口列表

建议布局:
  ┌─────────────────────────────┐

  │  📥  导入数据                │
```

### 推荐的设计 Token 系统

为了确保未来的一致性，建议在 Ardot 中定义以下变量：

```javascript
// 颜色变量
colors:
  dark-bg: "#0F0F0F"           // 页面背景
  dark-card: "#1A1A1A"         // 卡片/组件背景
  dark-border: "#2A2A2A"       // 边框/分割线
  text-primary: "#FFFFFF"      // 主要文字
  text-secondary: "#9CA3AF"    // 次要文字
  coral-default: "#D4724A"     // 橙色主色
  coral-light: "#E8845F"       // 橙色浅色
  coral-dark: "#C0623A"        // 橙色深色

// 圆角变量
radius:
  small: "8px"                 // 小元素（按钮、输入框）
  medium: "16px"               // 卡片
  large: "20px"                // 大卡片/弹窗
  pill: "9999px"               // 胶囊形状（标签、导航）

// 阴影变量
shadow:
  card: "0px 4px 12px rgba(0,0,0,0.5)"           // 卡片默认
  card-elevated: "0px 6px 16px rgba(0,0,0,0.6)"   // 浮起卡片
  button: "0px 4px 12px rgba(212,114,74,0.4)"     // 按钮阴影
  glow-orange: "0px 0px 8px rgba(232,132,95,0.3)" // 橙色光晕

// 间距变量 (8px 基准)
spacing:
  xs: "4px"
  sm: "8px"
  md: "16px"
  lg: "24px"
  xl: "32px"

// 字体大小
font-size:
  display: "28px"  // 页面标题
  h1: "22px"       // 区块标题
  h2: "18px"       // 卡片标题
  body: "16px"     // 正文
  caption: "14px"  // 辅助文字
  small: "12px"    // 标签/时间戳
```

## 🔧 Ardot 手动操作指南

由于批量编辑工具暂时遇到格式问题，你可以按照以下步骤在 Ardot 中手动实施关键优化：

### 步骤 1: 添加卡片阴影
1. 选择 **首页** 的三个卡片 Frame (`2:11`, `2:15`, `2:175`)
2. 右侧面板 → Effects (效果) 部分
3. 点击 "+" 添加新效果
4. 选择 **Drop Shadow (投影)**
5. 设置参数:
   - Color: `#000000` (Opacity: 50%)
   - Offset: X=0, Y=4
   - Blur: 12
6. 对 **个人中心** 的两个卡片重复此操作

### 步骤 2: 增强 AI 洞察卡片
1. 选择 AI 洞察区域的 Frame (`2:175`)
2. **Fill (填充)** 部分:
   - 删除现有 Solid fill
   - 点击 "+" → **Gradient (渐变)**
   - Type: Linear
   - Angle: 180° (从上到下)
   - Stop 1: Position 0%, Color: `#1A1A1A` at 90% opacity
   - Stop 2: Position 100%, Color: `#5B3425` at 15% opacity
3. **Effects (效果)**部分:
   - 修改已有 Drop Shadow 或新建
   - Color: `#E8845F` at 20% opacity
   - Offset: Y=6, Blur: 16

### 步骤 3: 优化保存按钮
1. 选择 **新建笔记**页面的保存按钮 (`2:49`)
2. **Effects (效果)**部分:
   - 添加 Drop Shadow
   - Color: `#D4724A` at 40% opacity
   - Offset: Y=4, Blur: 12

### 步骤 4: 创建设计变量（一次性设置）
1. 顶部菜单 → **Variables (变量)** 或按快捷键
2. 创建新的 Variable Set (变量集)，命名为 "Think Grid Design System"
3. 按照上文"设计 Token 系统"逐一添加变量
4. 未来所有组件都引用这些变量，方便一键换肤

## 📝 备注

1. **性能考量**: 阴影效果会增加渲染开销，但现代手机 GPU 完全可以轻松处理这些简单的 shadow
2. **可访问性**: 所有优化都保持足够的颜色对比度（WCAG AA 标准），不会影响视力障碍用户
3. **品牌一致性**: 橙色/珊瑚色的使用贯穿始终，强化"思格"的品牌记忆点
4. **后续扩展**: 如果未来需要支持亮色模式（Light Mode），只需替换设计 Token 的值即可
